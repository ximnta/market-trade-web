# market-trade-web

Angularjs app

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
$ git clone https://github.com/ximnta/market-trade-web.git
$ cd market-trade-web
$ npm install
$ npm start
```
## Build Production Distribution

$ npm run build-prod

The app should be running on [localhost:5000](http://localhost:5000/).
